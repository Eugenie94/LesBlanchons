Les Blanchons

Histoire           Description           Débats autour de la chasse aux blanchons


Dans les médias, on appelle souvent les blanchons « bébés phoques », ces derniers jouissant d'un statut particulier dans l'opinion publique et les campagnes publicitaires animalistes. Le blanchon est une espèce protégée.
